# v1.3.1 — DNS providers UI screenshots

Docs-only patch. No code changes since [v1.3.0](v1.3.0.md).

## What changed

- [`docs/screenshots/settings-dns-providers.png`](../screenshots/settings-dns-providers.png) —
  real capture of **Settings → DNS providers** replaces the 1×1
  placeholder. Shows the Cloudflare + AWS Route 53 cards, the
  Configured / Enabled badges, the `__UNCHANGED__` secret-field
  placeholder with Edit button, and the trust-boundary callout
  about credentials flowing through Caddy's admin API.
- [`docs/screenshots/host-form-dns-provider-dropdown.png`](../screenshots/host-form-dns-provider-dropdown.png) —
  real capture of the **host edit form** with DNS-01 selected and
  both providers (Cloudflare, AWS Route 53) available in the
  **DNS provider** dropdown.

Both images were previously 68-byte 1×1 transparent placeholders
carried over from v1.3.0-beta so `mkdocs build --strict` would pass
without broken image references. The capture checklist entries in
[`docs/screenshots/README.md`](https://github.com/cmos486/argos-edge/blob/main/docs/screenshots/README.md)
are now ticked.

## Upgrade

```bash
cd argos-edge
git pull
```

Nothing else changes. Running stack untouched.
